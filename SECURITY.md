# Security Policy

This repository is public and exploratory. Please do not submit secrets, credentials, private account metadata, or sensitive research artifacts in public issues or pull requests.

## Reporting Sensitive Issues

If you find a security issue, contact:

**tom@tombudd.com**

Please include:

- A clear description of the issue
- Steps to reproduce, if safe
- The affected file or behavior
- Whether any secret, token, credential, private backend metadata, or private data is involved

## Do Not Publish

Do not publish:

- API tokens or provider credentials
- Private quantum provider account details
- Private system internals
- Unredacted logs from non-public systems
- Personal data
- Unredacted attack chains against non-public systems

## Scope

This repository should contain toy circuits, synthetic data, simulator outputs, public methodology, and carefully qualified exploratory notes. If a private artifact appears here by mistake, report it immediately.
